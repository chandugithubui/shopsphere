# Concepts Learned

## Software Requirement Specification (SRS)

Learned that the SRS is a technical blueprint describing how the software should behave and be implemented.

---

## Difference between PRD and SRS

PRD answers:

"What should we build?"

SRS answers:

"How should we build it?"
